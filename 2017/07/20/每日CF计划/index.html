Codeforces Round #424 (7/8)

  • CF 831A 水
  • CF 831B 水
  • CF 831C 随便取一个 $b_0$，固定位置，其他所有的值就确定了。所以枚举下固定的位置，用 map 维护状态和去重。 复杂度 $O(n^2\log n)$
  • CF 830A 易证明结论，选取的钥匙必定是连续的一段，且钥匙按照位置顺序依次选取时，最大时间最短。因此就可以 $O(n^2)$ 的一个暴力。
  • CF 830B 简单拿个树状数组维护一下每个位置前面有多少张牌，记录下牌序，直接走一遍就可以了。trick 是牌有重复编号。
  • CF 830C 列一下式子会发现是 $d\sum \left \lceil \frac{a_i}{d} \right \rceil \geq k + \sum a_i$ 求 $d$ 的最大值，对于每个$a_i$ ，$\left \lceil \frac{a_i}{d} \right \rceil$ 的取值最多有 $\sqrt{a_i}$ 个。所以用$n/i$ 到 $n/(n/i)$ 分块枚举一下。对每个区间求一下 $d$。二分的话抠一下常数也能过。
  • CF 830D dp。考虑 $f(i,j)$ 表示第 $i$ 层，能够选出 $j$ 组互不相交的简单路径的方案数。对于每一层，令 $l$ 表示从左子树选出的路径组数， $r$ 是从右子树中选出的路径组数。枚举所有的 $l,r$ 组合，讨论是否选用根节点，根节点直接连在左右子树，和根节点直接连接两侧。可以得到转移。dp 的第 $2$ 维会到 $2^i$ 的范围。但是由于我们只需要知道 $f(k,1)$ ，由转移方程知道，每一个 $j$ ，最多向下影响到$j-1$。所以第二维实际只需要知道前 $k$ 个值。因此 $O(k^3)$。这个题卡常。

Codeforces Round #419 (6/7)

  • CF 816A 水题。
  • CF 816B 花式前缀和。
  • CF 815A 固定第一行，就唯一确定了所有的情况，暴力。
  • CF 815B 遇事不决打表找规律，比赛中不应该那么早就放弃治疗的，完全可以打打表？题解中有一个十分优雅的做法。大意就是可以拆成两个只包含加法的部分。
  • CF 815C 套路依赖背包。不过竟然可以用巧妙的实现，把这个朴素 dp 变成 $O(n^2)$，官方题解反而傻逼了（？）。看完 Um_nik 的证明和实现，还有这种操作.jpg?
  • CF 815D 考虑固定 $c$，后面就是一个区间取 $\max$，区间求和，用吉老师线段树就轻松秒过了啊（根本不用像题解那么麻烦的观察性质）。Um_nik 一个直接的做法：用容斥原理求三个阶梯形立方体的体积交，用前缀和处理出每个维度$x,y,z$ 的对应关系 $x(y),x(z),y(x),y(z),z(x),z(y)$，后面全都以以枚举 $x$ 为例，单个维度很容易求：$\sum_{x=1}^{p} ry(x)$， 两个维度交也易求：$\sum_{x=1}^{p} y(x)z(x)$。三个维度交比较麻烦，画图后发现分情况讨论：对于固定的 $x$ 如果$y(x),z(x)$构成的矩形在 $z-y$ 平面对应的阶梯之内，即$z(y(x))>y(x)$直接求得 $y(x)z(x)$ 就可以，否则可以把其拆成两部分的和， $z(y(x))y(x)+ f(z(x))-f(z(y(x)))$ ，前一项是矩形面积后一项是阶梯面积，其中$f(x)$ 是函数 $z(y)$ 的前缀和。

Codeforces Round #441 (8/8) DONE

  • CF 876A xjb gao
  • CF 876B 根据所有数模 $m$ 得到的剩余类分组，找到大于等于 $k$ 的组。
  • CF 875A 爆搜一下 $n-81$ 到 $n$ 之间的数。
  • CF 875B 冷静一下就不会像比赛中那样吃那么多 shit。考虑每次操作后的答案就是总的 $1$ 个数减去最长后缀 $1$ 的长度，后缀是单调的，暴力维护后缀。
  • CF 875C $2-SAT$ 。我们很容易根据上下相邻的两个串必须字典序有序来判断出字符之间选择大小写的关系，即通过最长公共前缀后一个字符来判断大小写关系，如果存在上面字符串包含下面的字符串则代表无解，特判输出。
  • CF 875D 做法很多。考虑答案总数减去不合法的情况。不合法的情况即一大串数或起来等于最大值。于是我们考虑每一段都在其最靠左的最大值的位置产生贡献。于是可以先用单调栈处理出每个的区间。还可以拆位找到每一位 $1$ 位置的有序序列。于是就可以对于最大值的每一个 $0$ 位拆位考虑缩小区间。$O(n\log n\log A)$。
  • CF 875E 考虑二分一个距离。考虑值域为布尔类型的 $f(i,j)$ 为当前送到第 $i$ 个人，另一个人在第 $j$ 个位置是否可行。初始状态只有 $s_1,s_2$ 两个点为 $1$，转移时对新的目标位置及其左右能延伸到的最大距离之内查找是否有 $1$，找到后对以外的位置区间置 $0$. 将新的位置设为 $1$ ，否则就不可达成。这个做法我写的常数巨大 1900ms 卡过。别人为什么都是 500 ms 左右啊？
  • CF 875F 给一个很大的二分图，边上有权，但是其中一边的点度等于 $2$，且这个节点连的两条边权值一样，求最大权匹配。由于有一边的度一直为 $2$，且权是以点为基础的。因此我们考虑把所有的男生当作点，女生当作边。做最大生成树。可以注意到只要联通块是棵树，那么我们可以随意往里面加边，每个边一定都能找到一个顶点归附，如果形成了一个环，则整个连通块都被固定了，于是这个题就变成了求最大生成一个最多有一个环的环林。Kruscal 。
