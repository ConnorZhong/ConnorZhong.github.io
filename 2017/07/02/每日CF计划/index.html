暑假每天坚持做 CF！

Codeforces Round #422 (5/6)

  • CF 822A 水题，阶乘不会爆 int。
  • CF 822B 暴力。
  • CF 822C 分别用 $l$ 和 $r$ 来排序，移动双指针时刻保持 $r < l$，这样维护所有当前遇到过的最小花费，移动 $l$ 时更新一下答案就行了。
  • CF 822D 有点卡常，可以用筛数法维护转移，$O(r\log\log r)$。可以证明当拆分为最小的质因数时，答案最优，所以可以进行优化，优化后的常数很棒。
  • CF 822F 其实是一个题面杀的构造题，很容易发现每个点最优的结果应该是 $2/\deg$，$\deg$ 表示该点的度。由于是树所以点与点之间其实不互相影响，所以简单的dfs构造一下就可以了。

Educational Codeforces Round 24 (6/7)

  • CF 818A 水题，推公式。
  • CF 818B 模拟一下，需要小心细节。
  • CF 818C 各个方向求一个前缀和，$O(1)$ 验证一下每一个沙发。
  • CF 818D 其实模拟一下就好了，维护一下每个时刻有多少车，更新的时候检查一下合法性。最后筛掉不合法的就可以了。
  • CF 818E 对 $k$ 分解质因数，后面处理的时候只用 $k$ 的质因数，其他的可以忽略，枚举一下 $x$，发现 $y$ 是单调的，因此拿个 map 维护一下当前遇到的相应质因数个数。一路坐过去就可以 $O(n*\log k)$。
  • CF 818F 结论易猜，就是选 $x$ 个节点出来，与剩下的 $n-x$ 个节点连接做桥。所以表达式 $2*x \leq (n-x)(n-x-1)/2$ 零点附近的两个值取得答案。二分以下，或者三分一下。

Codeforces Round #420 (5/5)

  • CF 821A 水题。
  • CF 821B 枚举下每一个 $y$，求出直线下方的最大的 $x$。
  • CF 821C 用栈模拟一下，一旦顶部不匹配，就把栈清空模拟打乱，标记一下打乱的值。重复下就做完了。
  • CF 821D 其实蛮套路的一个虚点最短路。当时第一反应是瞎 jb 缩点套 BFS。int 数组写成了 bool，于是失了智地交了 $16$ 发啊。
  • CF 821E 套路矩阵快速幂，其实没什么好讲的，检查是否爆 int 时一定要对每一个 int 逐行检查。
